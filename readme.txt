== Installation ==

= Using The Wordpress Dashboard =

1. First go to the 'Add New' Plugin Dashboard
2. Select `English-To-Bengali-Number-Convert.zip` file from your computer
3. Simply Upload
4. Activate the plugin via the Plugin Section of WordPress Dashbaord
5. After Activate the plugin Automatically Convert English to Bengali.

= Using FTP =

1. Extract `English-To-Bengali-Number-Convert.zip` to your computer
2. Upload the `Bengali-Date-Time-Numbers` directory to your `wp-content/plugins` directory
3. Activate the plugin on the WordPress Plugin Dashbaord
4. As this plugin supports All theme.
